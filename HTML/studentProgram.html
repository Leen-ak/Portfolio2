Overview

The **Student Program** is a Java-based application designed to manage student information efficiently. It features a **simple GUI** and allows students to **log in, view grades, and generate transcripts**. The application stores student data using **file handling (reading/writing to files)**.

Features

  • Student login and authentication system.
  • Stores student information, courses, and grades in files.
  • Generates GPA calculations and academic transcripts.
  • Simple and intuitive Java GUI interface.
  • Data persistence through file handling (text or CSV files).

Technologies Used

Programming Language: Java

Frameworks & Libraries: Java GUI

Data Storage: File Handling (TXT, CSV)
