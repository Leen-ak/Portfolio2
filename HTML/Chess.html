♟ The Chess Gambit

Overview

The Chess Gambit is an interactive chess game that allows players to compete against an AI opponent or challenge a friend in multiplayer mode. Designed with a focus on strategy and user experience, this project provides a clean, responsive interface and a well-structured game logic.

Features

  • ✔️ Single-Player Mode with AI opponent.
  • ✔️ Multiplayer Mode for local two-player games.
  • ✔️ Move validation to prevent illegal moves.
  • ✔️ Check & Checkmate detection.
  • ✔️ Game history tracking.
  • ✔️ Responsive design for different devices.

Technologies Used

Frontend: HTML, CSS, JavaScript

Backend (if online): Node.js & WebSockets

AI Engine: Minimax algorithm with alpha-beta pruning
